# Habit Tracker Template (Google Sheets)

A clean, formula-driven habit tracker for tracking daily habits and visualizing progress.

## Overview
This monthly habit tracker lets you track daily habits using checkboxes and automatically view your progress through charts and summaries.

## Get Your Own Copy
You can copy this tracker to your own Google account by opening the link below and selecting  
**File → Make a copy**.

https://docs.google.com/spreadsheets/d/1gWKk4MilY0uMoGJXDQTRNttGmb6l4EPLxFBwVQWMHAw/copy

## How to Use
1. Open the link above
2. Click **File → Make a copy**
3. Set the year and month
4. Customize habit names and goals
5. Check habits daily

## Notes
For the best experience, use the Google Sheets version.  
Downloading to Excel may affect checkboxes and layout.
